<h1 align="center">Abstract Factory</h1>

<p>Este sistema utiliza o padrão de projeto Abstract Factory para alcançar a flexibilidade na criação de objetos relacionados.
  O padrão Abstract Factory fornece uma interface para criar famílias de objetos relacionados sem especificar suas classes concretas.</p>

##

#### Exemplo 1
<p>O exemplo consiste em um sistema de criação de veículos, incluindo carros e motos, com diferentes variantes, como carros esportivos, carros familiares, motos esportivas e motos padrão.
  O objetivo do sistema é criar esses veículos de forma flexível, permitindo a criação de diferentes tipos de veículos com base em diferentes fábricas.</p>
